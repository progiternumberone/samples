Frankenstein Application

Frankenstein is an application that will provide reusable functionality and data entities for faster app development.

View in GIT

How to add to your project

Include Frankenstein by adding the following script tags on your HTML page.

Files to create in your application

  • /js/frankenstein-config.js - configuration of location of your JS and template folders
  • /js/businessLogic.js - application controller that ties your business logic to the controls and entities provided by Frankenstein
  • /js/routes.js - set the URL paths to functions in businessLogic.js
  • /js/layout.js - controls your applications layout
  • /template/... - folder with HTML templates that will render in your application's UI
